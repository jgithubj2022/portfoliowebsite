## Project Inspiration

For this project, I wanted to showcase my love for the Xbox as a kid and the nostalgia I found in the Xbox 360 avatar selection screen. Inspired by those memories, I integrated that experience into my personal portfolio website to blend my hobbies with my passion for computer science.

I screen-recorded my Xbox avatar's idle animation, removed the background using chroma keying in Adobe Premiere Pro, and designed a retro television inspired by Panasonic box TVs from the early 2000s. After rendering the animation in Blender, I exported it as a video and synchronized the loop to create a seamless animated start screen :)
